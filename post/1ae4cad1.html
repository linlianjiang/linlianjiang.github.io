16_Tutorial on Variational Autoencoders

Created2023-01-27|Updated2023-07-10|paper learningOther
paper code

1. Introduction

  1. Generative modeling有一个P(X)的分布，X在一个高维空间，且P（x）是可以计算的

  2. 有监督学习可以将其分为两类模型：判别式模型和生成式模型。简单地说，判别式模型是针对条件分布建模，而生成式模型则针对联合分布进行建模。

  3. 生成模型：p(y|x)=p(x,y)/p(x),你给我x和y，我先假设二者的联合分布是p(x,y)，然后根据贝叶斯公式推出y（入上面的公式），p(x,y)需要指定它两满足什么联合分布，典型的例子就是朴素贝叶斯了

    判别模型 ：f(x)=y，你给我x，我给你拟合一个函数出来使得f(x)=y。 联合概率和条件概率的区别

  4. 目的： 从原始图像数据库 合成新的、没见过的图像

  5. 先训练一个AE：学习一个我们可以从中采样的模型 P，使得 P 与 Pgt 尽可能相似

  6. VAE就是在自编码器模型上做进一步变分处理，使得编码器的输出结果能对应到目标分布的均值和方差

1.1 Preliminaries: Latent Variable Models

  1. 输入字符5的左半部分，则右半部分不能包含0的左半部分，否则该字符显然看起来不像任何真实的数字，各自有各自的分布，输入一个猫不可能变成狗的分布

  2. latent variable： 模型在为任何特定像素赋值之前首先决定生成哪个字符

  3. 将图片传给神经网络的时候，神经网络自己对这张图片进行特征提取，这是神经网络自己内部的行为，我们并不知道它能提取到什么样的特征，我们只能告诉它你以目前这种方式提取的特征所获得的分类/生成图片结果，是正确/错误的，并告诉它有多正确/多错误。它根据我们的指示所提取到的特征，是以神经网络自己内部的形式对图片进行编码(encode)的结果，而编出的码，就是神经网络对Latent Variable进行的模拟。Latent Variable代表的是图片背后的信息，比如数字几，笔画粗细，数字的角度等等。但是需要注意的一点是，以上仅仅为举例，现实中人类往往并不能直接通过这些编码获得有用的信息，只有神经网络自己才看得懂这是啥。

  4. 在我们的模型绘制任何东西之前，它首先从集合[0,…,9] 中随机采样一个数字值z , 然后确保所有笔画都与该字符匹配

  5. z 被叫做latent， 因为仅给定模型生成的字符，我们不一定知道生成该字符的隐变量是什么，所以需要训练

  6. P(X) =∫P(X|z; θ)P(z)dz

    • θ是 X|z 概率分布的参数
    • 一个随机变量 X， θ为分布参数，如果一组观测样本值为 x1,…xn，那么我们可能会问“在所有可能的 θ 值中，什么值将使得获得观测样本集 x1,…xn 的可能性最大
    • 这个问题的目标是建立一个模型，然后不断的优化 θ , 使得在这个模型下，已知数据集 X 发生的可能性最大，这就是Maximum Likelihood(最大似然)

  7. VAE中，一般用 Gaussian 分布：P(X|z; θ) = N (X| f (z; θ), σ2 ∗ I)

2. Variational Autoencoders

  1. VAE的两个问题： 如何定义 Z 以及如何处理z上的积分

  2. latent variable在手写数据集中，z 不仅需要选择数字，还需要选择数字绘制的角度

  3. 如果f（z；θ）是一个神经网络，那么我们可以想象网络使用其前几层将正态分布的z映射到具有正确统计的潜在值（如数字身份、笔划权重、角度等）这里是 latent attributes：

  4. 然后，它可以使用后面的层将这些潜在值映射到完全渲染的数字

  5. P(z) = N (z|0, I)

2.1 Setting up the objective

  1. VAE的关键思想是尝试对可能产生 X 的 z 值进行采样，并从中计算 P(X)
  2. 需要一个可以取 X 值的新函数 Q(z|X)，并为我们提供了可能产生 X 的 z 值上的分布

3. AE

4. VAE

  1. VAE 解决了AE在content generation上的问题，我们希望产生新的数据

  2. VAE训练的步骤：

    • input通过encoder在latent space得到分布
    • 从latent space采样
    • decoder重建采样点
    • 整个过程可以反向传播

  1. latent space 预期的规律性： continuity （连续性， latent space中的两个接近点一旦解码，不应给出两个完全不同的内容）和 completeness （完整性，对于所选的分布而言，一旦解码，从 latent space 采样的点应给出“有意义的”内容）

  1. 必须同时将协方差矩阵和编码器返回的分布的平均值进行正规化。实际上，通过强制执行分布接近标准正态分布（居中和减少）来完成此正则化。这样，我们要求协方差矩阵接近 identity ，防止 punctual distributions 以及接近0的均值，以防止编码的分布与彼此之间的距离太远。
  2. 我们可以观察到，正则化获得的连续性和完整性倾向于对潜在空间中编码的信息产生“梯度”。例如，在两个来自不同培训数据的编码分布之间的潜在空间的点应该是在给出了第一个分布的数据和第二个分布的数据之间进行解码的在两种情况下，自动编码器都可以通过自动编码器进行采样。
