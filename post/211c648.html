21_CVPR_View-Guided Point Cloud Completion

Created2022-11-07|Updated2022-11-07|paper learningMultimudal
paper code

0. Abstract

  1. 与大多数使用形状先验直接推断缺失点的现有方法不同，引入 ViPC（视图引导点云完成）来解决此任务，该方法从额外的单视图图像中获取缺失的关键全局结构信息

    多模态方法

1. INTRODUCTION

  1. 点云补全： 编码器将不完整的输入点云传输到特征空间，然后解码器通过将特征传输回欧几里得空间来重建完整的点云
  2. 单模态点云补全缺点：1）由于以下因素推断缺失点存在很大的不确定性由于可用信息量有限，2）点云是非结构化数据，加上固有的稀疏性，很难确定空白的3D空间是由固有的备用性还是不完整性引起的。
  3. 关键： 如何有效融合局部点云提供的位姿和局部细节信息和单视图图像提供的全局结构信息
  4. crossmodality：（信息来自图像和点云两种模态）“cross-level”：（局部细节和全局结构是来自不同层次的信息）
  5. crossmodality：通过从单视图图像重建coarse point；cross-level通过称为“动态偏移预测器”的网络授权的差分细化策略来解决，也就是coarse to fine

2. Related Work

现有的点云补全方法通常可以分为三种类型的方法：基于几何的、基于对齐的和基于学习的

Geometry-based Methods

  1. 通过先前的几何假设直接从观察到的形状部分预测不可见的形状部分
  2. 一些方法通过生成平滑插值来局部填充表面孔，例如拉普拉斯平滑和泊松表面重建
  3. 其他方法检测模型结构中的规律性并重复它们以根据已识别的对称轴预测缺失数据

Alignment-based Methods

  1. 在形状数据库中检索与目标对象相似的相同模型，然后将输入与目标模型对齐并完成缺失区域

Learning-based Methods

  1. 构建一个参数化模型来学习一个形状的不完整和完整点云的两个特征空间之间的映射

3. Method

3.1 Overview

Problem Definition

  1. 输入图像包含缺失部件（missing shape part）必要的结构信息

  2. 目标是恢复一个由两部分组成的 3D 形状，即 S = { ^S0, ΔS}，其中 ^S0 是接近输入部分形状 S0 的形状部分, ΔS 是未知的缺失形状部分。形式上，我们将 M(S) 表示为模态 M 中形状 S 的表示，即 M = P 表示点云，M = I 表示图像。任务的输入可以表述为：

    • The partial point cloud P(S0).
    • The single-view image I(SI ), where SI refers to the observed shape from the view of image.

Pipeline

  1. 第一阶段称为 Modality Transfermaps I 到 P 中点云的coarse表示，然后将重建的点云与 3D 空间中的输入部分形状 S0 对齐。然后称为 Part Filter 的第二阶段从两个对齐的点云生成一个coarse point cloud
  2. 该阶段还对主要来自输入部分点云的点和主要从输入图像 I(SI ) 重建的其他点进行了区分。我们称前者为 Fine Part，后者为 Coarse Part
  3. 最后，称为零件细化的第三阶段将粗零件和精细零件作为输入，并生成完整的更高质量的点云。在这个阶段，我们主要细化 Coarse 的形状，但使用 Fine 部分作为细化的约束。这比在没有任何约束的情况下对所有点进行细化的全局细化可以获得更好的结果

3.2 Modality Transfer

  1. 模态转移
  2. 编码器将输入图像映射成一个潜在空间向量，解码器输出一个Nr×3矩阵，其中每一行代表一个点的笛卡尔坐标
  3. 解码器应用一系列反卷积层并将输出展平以生成 Nr 个点坐标。来自编码器每一层的特征图也被保留下来，作为此阶段后续过程的附加指导
  4. 从单视图图像重建 Pr (SI ) 后，我们将其与输入部分点云对齐。

3.4 Part Refinement

3D Guidance

  1. 之前已经有方法验证了基于现有形状和先验知识的补全是一种可行的方式。例如，飞机的对称性可以帮助我们在一侧完成缺失的机翼。因此，我们通过在部分点云 P(S0) 和重建的 Pr (SI) 上应用通用编码器 PointNet 来提取全局特征

2D Guidance

  1. 单视图对coarse part恢复结构和几何细节提供了引导
  2. 为了增加局部点之间的变化并防止为不同的点预测相同的偏移量，受 FoldingNet 的启发，需要生成 2D 网格并重新整形为一个特征向量作为 fgrid 以增强轻微的干扰。
