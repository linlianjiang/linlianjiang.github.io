Title: StructureNet: Hierarchical Graph Networks for 3D Shape Generation

  1. Structure
  2. 部件的连续变形以及结构或离散的改变，增加、删除或修改形状成分和组成结构
  3. 这种对象结构通常可以组织成由组成对象 parts and relationships 组成的层次结构，表示为 n-ary 图的层次结构
  4. 介绍 StructureNet，层次图网络：
    • 直接对表示为这样的 n-ary graphs的形状进行编码
    • 在大型和复杂的shape families 上进行有力的训练
    • 生成多种逼真的结构形状几何图形
  5. 从图神经网络的最新进展中获得灵感，提出了一种n-ary图的序不变编码
  6. 在网络训练中，综合考虑了零件的几何形状和零件间的关系
  7. 实现多种结构感知的几何处理应用，包括形状生成和插值，形状编辑
  1. StructureNet是一个 hierarchical graph network，它产生一个统一的潜在空间来编码具有连续几何和离散结构变化的结构化模型
  2. 如上图所示：将一个未注释的点云(左)和一个未注释的图像(右)投影到学习过的潜在空间中，从而产生作为图的层次结构的语义分割的点云。潜在空间的形状插值也产生结构化点云(上)，包括它们对应的图形(下)。边对应于由我们的方法建模的特定零件关系。为了简单起见，这里我们只显示没有层次结构的图。注意椅子的底座是如何通过功能上合理的中间配置而变化的，或者椅子的靠背从一个普通的靠背过渡到一个有扶手的靠背。

1. Introduction

  1. 可以融合所有这些多样性，并能够编码，例如，椅子有或没有扶手，椅子有四条腿或旋转底座，以及高或低背，细或粗腿
  2. 将形状表示为 structured objects，该对象由根据 part-level connectivity和inter-part relationships 的部件集合组成
  3. 如上图所示：shape part组织成 n-ary 层次结构

structurenet 特性部分没看懂

  1. hierarchical graph network:
    • 提取原始输入，包括点云、图像、部分扫描，以获得其结构
    • 以参数化形式或基于一组训练形状的点云形式创建新的形状
    • 支持 source 和target shape之间的 structure-aware interpolation（结构感知插值），同时显示拓扑和几何变化
    • structure-aware object manipulation（结构感知对象操作），以智能地修改零件或更换零件。例如，在图1中，输入源点云和目标图像首先被独立地提取，然后直接插值到潜在空间中，该潜在空间是在5K椅子上从PartNet数据集中学习的
  2. Contributions:
    • 为n-ary part graph的形状层次结构引入编码，该编码足够通用，允许在一个类别中对形状进行一致的层次表示
    • 学习 encode 和decode sibling part 之间丰富的几何关系（例如，邻接、对称）， 用层次图中的边表示，用图神经网络约束真实形状的生成
    • 开发一个生成性模型 StructureNet，允许对具有各种有效几何和结构变化的长方体结构和点云进行形状合成

2. RELATED WORK

  1. 工作主要涉及结构感知形状表示，它超越了局部几何，通过形状零件之间的排列和关系来描述形状，以及3D形状生成模型，旨在对3D形状的变化进行建模，并合成新的3D形状

Structure-A ware Shape Representations

这些方法通常以生成 scene hierarchy 为目标，依靠对象检索来完成场景，而我们主要关注 object-level structure synthesis 合成以及相应的 part-level geometric details

3D Deep Generative Models

  1. wang[2018a]的方法是：首先学习在一组分割的形状上使用GAN合成基于体素的形状结构和零件和标签，然后通过自动编码器优化每个零件的几何结构

  2. 形状结构可以从分布中采样和生成。但是，所需的二进制对称层次结构可能会在节点中引入任意顺序，并使层次结构不一致跨形状

  3. 如上图所示：与GRASS相比，使用StructureNet(colored box)和GRASS(blue box)比较了几对椅子之间的插值。在被两种方法(标记为“source”和“target”)重构后，我们在测试集的最左边和最右边显示的形状之间进行插值。我们的插值使用了大量较小的结构变化来达到目标形状

  4. StructureNet允许直接处理 n-ary graph，并进一步explicitly（显式地）为图中各部分之间的关系建模。因此，它可以在更大的数据集上稳健地训练

Neural Networks for Graph Structured Data

  1. 要生成的形状结构，本质上是零件的布局和它们之间的关系，可以表示为 n-ary graph的层次结构
  2. Li[2017]对树形结构进行编码使用递归神经网络(RvNN)依次折叠图的边缘
  3. 本文设计了一种新的图形编码框架，将RvNN与 graph convolutions（图形卷积）相结合，以处理表示为图的层次结构的图形
  4. 在tree-like graph生成方面，自然语言处理和程序合成做了大量的工作，但这些工作大多局限于树，不能处理更一般的图

3. OVERVIEW

  1. 将形状部件的层次结构表示为一个n-ary tree，其中每个节点都是一个部件或部件组装
  2. 部件的几何关系，例如对称性和邻接性，通过树的兄弟之间的附加边来捕获，形成兄弟之间的图
  3. 上图给出了一个示例。每个节点包含有关零件几何形状的信息，总捕获形状的几何形状。在给定的类别中，如椅子，形状往往具有一致的零件层次结构，如PartNet数据集所示
  4. 例如，大多数椅子在最高层次上自然分解为靠背、座椅和底座。形状表示中的n-ary tree可以直接捕获这种层次结构，从而为我们提供了在给定类别的形状之间具有高度一致性的形状表示

4. A HIERARCHY OF GRAPHS FOR SHAPE STRUCTURE

  1. 引入了一种图形表示方法，该方法能够同时捕捉图形的几何和结构，适合于我们的分层图网络处理。我们假设我们处理的形状可以分解成一组有意义的部件
