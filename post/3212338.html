PyTorch(三)：线性回归深度学习实现

Created2022-02-04|Updated2022-03-12|DeeplearningPytorch

1. 生成数据集

  1. 使用 help(d2l.synthetic_data) :

    synthetic_data(w, b, num_examples)
    Generate y = Xw + b + noise

2. 读取数据集

  1. 调用框架中现有的API来读取数据
  2. 将features和labels作为API的参数传递，并在实例化数据迭代器对象时指batch_size

  3. 布尔值 is_train 表示是否希望数据迭代器对象在每个迭代周期内打乱数据

  4. 使用 data_iter 的方式之前相同。不同的是，这⾥我们使用 iter 构造Python迭代器，并使用next从迭代器中获取第一项

3. 定义模型

  1. 首先定义⼀个模型变量net，它是⼀个Sequential类的实例

  2. Sequential类为串联在⼀起的多个层定义了⼀个容器。当给定输⼊数据，Sequential实例将数据传⼊到第⼀层，然后将第⼀层的输出作为第⼆层的输⼊，依此类推（类似神经网络）

  3. 在PyTorch中，全连接层在Linear类中定义。将两个参数传递到nn.Linear中

4. 初始化模型参数

  1. 需要初始化模型参数。如在线性回归模型中的权重和偏置
  2. 这里，指定每个权重参数应该从均值为0、标准差为0.01的正态分布中随机采样，偏置参数将初始化为零
  3. 通过 net[0] 选择网络中的第⼀个图层，使用 weight.data 和 bias.data 方法访问参数。然后使用替换方法 normal_ 和 fill_ 来重写参数值

5. 定义损失函数

计算均方误差是 MSELoss 类，也称为平方 L2 范数。返回所有样本损失的平均值

6. 定义优化算法

  1. 实例化SGD实例时，要指定优化的参数（可通过net.parameters()从模型中获得）
    以及优化算法所需的超参数字典
  2. 小批量随机梯度下降需要设置 lr 值，这里设置为0.03

7. 训练

  1. 在每个迭代周期里，我们将完整遍历⼀次数据集（train_data），不停地从中获取⼀个小批量的
    输入和相应的标签

  2. 对于每⼀个小批量，进行以下步骤:

    • 调用 net(X) 生成预测并计算损失 l（正向传播）
    • 反向传播来计算梯度
    • 调用优化器来更新模型参数
    • 为了更好的衡量训练效果，计算每个迭代周期后的损失，并打印它来监控训练过程
