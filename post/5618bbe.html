18_SIGGRAPH Asia_Learning to Group and Label Fine-Grained Shape Components

Created2022-02-20|Updated2022-03-12|paper learningPoint Cloud
paper no code

0. Abstract

Title: Learning to Group and Label Fine-Grained Shape Components

1.introduction

  1. 为了实现强大的零件假设标记，我们采用三维卷积神经网络（CNN）从零件假设的体积表示中提取特征
  2. 为了学习不仅捕获局部零件几何图形，而且还捕获全局上下文信息的特征，我们设计了一个以两个三维体积比例作为输入的网络
  3. 局部尺度通过在其边界框内对零件进行体素化的特征提取，对感兴趣的零件假设本身进行编码
  4. 全局体积将整个形状的边界框作为输入，并使用两个通道对上下文进行编码，这两个通道对比了部分假设本身和其余部分的体积占用情况
  5. 该网络输出不同零件类别上零件假设的标记概率，用于最终标记的分割
  6. 为了完成整个形状的标记分割，我们制定了高阶条件随机场（CRF）来推断每个组件的最佳标记分配
  7. 我们在我们的多分量标记（MCL）基准数据集上验证了我们的方法。从ShapeNet和3D Warehouse收集多组件3D模型，并手动标记所有组件。与其他方法相比，我们的方法在对高细粒度组件进行分组和标记方面实现了更高的准确性
  8. Contributions:
      - 我基于预先存在的高细粒度组件的stock三维模型标记分割问题，并用零件假设生成和特征化的新解决方案来解决该问题
     - 提出了一种多尺度3D CNN，用于编码局部和上下文信息，用于零件假设标记，以及基于CRF的组件标记公式

2. RELATED WORK

3. Method

  1. 给定包含多个组件的3D模型（a），我们的方法首先使用自底向上的分组程序（b，第3.1节）对候选零件假设进行分层采样
  2. 然后将每个候选零件输入多尺度卷积神经网络，该网络预测其标签，并根据每个零件标签回归置信度得分（c，第3.2节）
  3. 根据置信度得分（d，第3.3节），使用高阶条件随机场推断每个成分的最佳标签

3.1 Generating part hypothesis

  1. 通过统计分析（图5），我们发现大多数语义部分在空间上是紧凑的，比如车门或车轮。因此，我们定义了一个称为中心距离的标准，用Ccenter（a，b）表示，来测量两个组件a和b之间的紧度。它测量两个组件的凸包重心之间的距离，并鼓励对空间上彼此接近的组件进行分组

3.2 Classifying and Scoring of Part hypotheses

  1. 我们训练一个神经网络对一个部分假设进行分类，并为其生成一个置信度分数，表示假设是一个独立语义部分的置信度
  2. 建立一个具有组件标签的多组件三维模型的训练数据集。然后，我们设计了一个多尺度卷积神经网络（CNN），该网络学习特征表示，不仅捕获局部零件几何，还捕获全局上下文

Input data representation

  1. 为了训练CNN模型进行零件假设标记，我们选择零件假设的体积表示作为输入。为了实现多尺度特征学习，我们在三个尺度中表示每个假设，包括基于其边界框体素化的局部尺度，一个全局尺度，它取整个形状的边界框的体积，并贡献两个通道。一个通道对零件假设本身的体积占用进行编码，另一个通道根据剩余零件的占用情况解释上下文。对于每个刻度，体积分辨率固定为30×30×30。为了避免所有形状之间的全局对齐，我们选择使用每个形状的许多可能方向进行训练。在实践中，我们使用每个形状的右上方向，并列举其四个标准方向（曼哈顿框架）

Data augmentation for balanced training

  1. 零件假设的分层分组可能会导致训练数据不平衡：对于包含少量部件（例如汽车后视镜）的语义类别，采样的数据不足。这将使我们的CNN模型在这些方面的训练不足。为了解决这个问题，我们选择基于训练数据中语义部分的基本事实，为实例不足的类别合成更多的训练数据。具体来说，我们追求两种数据扩充方式
    • 组件删除。给定一个基本事实语义部分，我们随机删除几个组件，并使用不完整部分作为训练示例。我们通常会移除多达30%的组件
    • 组件插入。给定一个基本事实语义部分，我们从相邻部分随机插入几个组件，形成一个训练示例。我们规定新增加的部件不超过原部件的30%

Network architecture

  1. 设计了一个多尺度卷积神经网络(multi-scale Convolutional Neural Networks)
  2. 该网络有三个towers，接收与上述local和两个 global channels相对应的输入
  3. 这些 towers：local, global and contextual
  4. 三个towers输出的特征图被连接成一个特征向量，然后馈送给几个完全连接的层(fully connected layers)
  5. 最后一个完全连接的层预测一个标签，并为输入部分假设回归一个分数

3.3 Composite Inference and Labeling

1. 给定采样部分假设的置信度分数，我们方法的最后阶段是为每个组件推断一个最优的标签分配
 2. 给定一个多组分三维模型，用M表示，该模型由一组组分C组成
