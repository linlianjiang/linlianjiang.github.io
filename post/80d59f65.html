Lecture 4:Edge Detection

4.1 Edge character

  • 边缘是图像强度函数快速变化的地方

4.2 Image gradient

  • 梯度方向
  • 边强度由梯度大小给出

4.3 Effects of noise

  • 有限差分滤波器对噪声有很强的响应：
    • 图像噪声导致像素看起来与周围的像素非常不同
    • 噪声越大，响应越强
  • 解决方式：
    • 平滑图像，通过迫使像素与它们的邻居不同(=噪声像素?)，让它们看起来更像邻居

  • Mean smoothing：
  • Gaussian (smoothing * derivative)：

4.5 Designing an edge detector

  • “最佳”边缘检测器的标准:
    • 良好的检测：最小化假阳性(检测噪声引起的假边)和假阴性(丢失真实边)的概率。
    • 良好的定位：检测到的边缘必须尽可能接近真实边缘
    • 单响应： 对于每个真边缘点，检测器必须返回一个点;也就是说，最小化真实边缘周围的局部最大值的数量

4.6 Sobel Operator

索贝尔算子

  • 使用两个3×3核与原始图像卷积来计算导数的近似值
  • 一个用于水平变化，一个用于垂直变化

4.7 Hough transform

霍夫变换

  • HT用来检测直线，图像中寻找线条
  • 注意：Hough变换只有在直线、圆和其他结构的参数方程已知的情况下才能检测到它们。
  • 在噪声和部分遮挡条件下具有较好的鲁棒性
  • 过程：
    • 我们希望找到构成直线的像素集。
    • 考虑一个已知坐标点 ，且许多条线通过坐标点 ，通过这一点的直线为，它们的共同之处在于它们满足某个参数集的方程（a,b）

Lecture 5:RANSAC

5.1 RANdom SAmple Consensus：

随机抽样一致性

  • 方法：我们希望避免异常值的影响，所以让我们寻找“内部异常值”，并只使用这些异常值。
  • RANSAC 过程:
    • 随机选择 a seed of group 作为变换估计的基础(例如，一组匹配点)
    • 从 seed group 计算变换
    • 找到这个转换的 inliers
    • 如果内嵌层数目足够大，则重新计算所有内嵌层上变换的最小二乘估计

  • 任务：评估最佳直线
    • 重复过程，直到找到最佳结果

  • RANSAC优点：
    • 适用于广泛的模型拟合问题
    • 实现简单，故障率计算简单
  • RANSAC缺点：
    • 计算参数的迭代次数没有上限；如果设置迭代次数的上限，得到的结果可能不是最优的结果，甚至可能得到错误的结果。RANSAC只有一定的概率得到可信的模型，概率与迭代次数成正比。
    • 它要求设置跟问题相关的阀值

5.2 Finding Corners

  • Design criteria
    • flat：所有方向都没有变化
    • edge:沿边缘方向无变化
    • corner:各方向都发生了重大变化

5.3 Harris Detector Formulation

  • Harris Matrix

    • 考虑一个轴对齐的角：

      • 梯度方向与x,y轴对齐
      • 如果λ任意一个都接近0，那么这就不是一个角，所以寻找两个角都很大的位置。

  • 考虑一个与图像轴不对齐的角：

    • 把M看作椭圆，它的轴长度由特征值决定，方向由R决定

  • 利用M的特征值对图像点进行分类

Lecture 6:Feature Descriptors

6.1 Scale Invariant Detection

规模不变检测

  • 方法：
    • 在区域(圆)上设计一个函数，它是“尺度不变的”(对应的区域也是一样，即使它们在不同的尺度上)
    • 对于图像中的一点，我们可以认为它是区域大小(圆半径)的函数

  • 取这个函数的局部最大值，获得最大值的区域大小应该与图像尺度共同变化
  • “良好”的尺度检测功能:有一个稳定的尖峰
  • 好的函数应该是对对比度做出反应的函数(强烈的局部强度变化)

6.2 Harris-Laplacian

  • 求局部最大值:
    • 空间哈里斯角感探测器(图像坐标)
    • Laplacianin scale

6.3 DoG

Difference-of-Gaussians

高斯分布在空间和尺度上的差异

6.4 SIFT

图像区域描述符

  • 过程
    • 使用与关键点的比例相关的模糊图像
    • 在关键点附近取图像梯度
    • 要成为旋转不变量，通过(-关键点方向)旋转梯度方向和位置
      • 现在我们已经抵消了旋转，并且有了在相对于关键点方向θ位置上的梯度
      • 我们也可以把整个图像旋转-θ，但那样会慢一些。
    • 将旋转的梯度放入其局部方向直方图
    • 最好的结果是每个直方图有8个方向柱

6.5 DoG and SIFT

  • HoGis通常用来描述整个图像,SIFT用于关键点匹配。
  • SIFT直方图是面向优势梯度的。HoG不是。
  • HoG梯度使用邻域容器进行标准化。SIFT描述符使用不同的尺度来计算多个描述符。

Lecture 9:k-means & mean-shift clustering

9.1 K-means clustering

  • 过程：
    • 随机选取k个簇
    • 把每个点分配到最近的中心
    • 将聚类中心更新为点的平均值
    • 更新，重复步骤2-3直到停止

  • 聚类结果：
    • 基于强度或颜色的K-means聚类本质上是对图像属性的矢量量化
      • 集群不需要在空间上连贯
    • 基于(r,g,b,x,y)值的聚类增强了空间一致性
  • 优点：
    • 找到最小化条件方差的集群中心(良好的数据表示)
    • 简单快速，易于实现
  • 缺点：
    • 需要选择K
    • 对异常值敏感
    • 容易出现局部极小值
    • 所有簇具有相同的参数(例如，距离度量是非自适应的)
    • 可能很慢:对于N个点，每次迭代是O(KNd)
  • Usage:
    • 无监督聚类
    • 很少用于像素分割

9.2 Mean-shift clustering

一种先进和通用的基于聚类的分割技术

  • 迭代模式搜索算法：
    • 初始化随机 seed ，window w
    • 计算W的重心(平均值)
    • 将搜索窗口移到平均值
    • 重复步骤2直到收敛

  • 所有数据点向Attraction basin移动
  • Attraction basin：所有轨迹都指向同一模式的区域

12.2 PCA

Principal Component Analysis（主成分分析）

数据降维算法

  • 场景
    • 原始数据样本有很多冗余和模式
    • PCA将数据样本表示为对主要成分的权重，而不是使用原始数据形式
    • 通过用权重表示每个样本，表示样本之间的差异
    • 这种最小表示使机器学习和其他算法的效率更高
  • 思想：
    • 将n维特征映射到k维上，这k维是全新的正交特征也被称为主成分，是在原有n维特征的基础上重新构造出来的k维特征
  • 原理：
    • 从原始的空间中顺序地找一组相互正交的坐标轴，新的坐标轴的选择与数据本身是密切相关的。
    • 第一个新坐标轴选择是原始数据中方差最大的方向，第二个新坐标轴选取是与第一个坐标轴正交的平面中使得方差最大的，第三个轴是与第1,2个轴正交的平面中方差最大的，依次类推，可以得到n个这样的坐标轴
    • 通过这种方式获得的新的坐标轴，大部分方差都包含在前面k个坐标轴中，后面的坐标轴所含的方差几乎为0
    • 于是，我们可以忽略余下的坐标轴，只保留前面k个含有绝大部分方差的坐标轴。相当于只保留包含绝大部分方差的维度特征，而忽略包含方差几乎为0的特征维度，实现对数据特征的降维处理
  • PCA by SVD
    • 输入：数据集 X = ｛X1, X2, …，Xn｝ 需要降到k维
    • 去平均值，即每一位特征减去各自的平均值
    • 计算协方差矩阵
    • 通过SVD计算协方差矩阵的特征值与特征向量
    • 对特征值从大到小排序，选择其中最大的k个。然后将其对应的k个特征向量分别作为列向量组成特征向量矩阵
    • 将数据转换到k个特征向量构建的新空间中

Lecture 15:Object detection

15.1 T(F) Positives(Negatives)

  • True Positive：真阳
    • 正样本被预测判定为正
    • 判断为阳性且判断正确情况
  • True Negative：真阴
    • 负样本被预测判定为负
    • 判断为阴性且判断正确情况
  • False Positive：假阳
    • 负样本被模型预测为正
    • 判断为阳性但判断错误情况，或称为误报
  • False Negative：假阴
    • 正样本被模型预测为负
    • 判断为阴性但判断错误情况，或称为漏报

15.2 DMP

Deformable Parts Model(可变形的零件模型)

  • 方法：
    • 手动选择一组零件
    • 为每个零件训练的特定检测器
    • 训练零件激活的空间模型
    • 评估零件激活的联合可能性
  • 优点：
    • 零件有直观的含义
    • 每个部件都可以使用标准的检测方法
    • 适用于特定类别
  • 缺点：
    • 部件需要手动选择
    • 语义驱动的部件有时没有一个简单的外观分布
    • 不能保证一些重要的部件没有被错过

Lecture 17: Motion

17.1 Optical flow

  • 定义：光流是指图像中亮度模式的明显运动
  • 目标：从光流恢复每个像素处的图像运动
  • 估计光流：给定两个后续帧，估计它们之间的视运动场 u（x，y），v（x，y）

  • 关键假设：
    • 亮度恒定性：同一点的投影在每一帧中看起来都相同
    • 微小运动：点不会移动太远
    • 空间一致性：点像其邻居一样移动
