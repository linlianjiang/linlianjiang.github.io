22_ECCV_SeedFormer Patch Seeds based Point Cloud Completion with Upsample Transformer

Created2022-11-07|Updated2022-11-07|paper learningPoint Cloud
paper code

0. Abstract

  1. 从partial observation中恢复 3D 对象的完整形状是一个具有挑战性问题

  2. 与之前基于 global feature 的方法不同，我们引入了一种新的形状表示，即 Patch Seeds，它不仅从部分输入中捕获一般结构，而且还保留了局部模式的区域信息

  3. 然后，通过将seed feature集成到生成过程中，我们可以从粗到细的方式恢复完整点云的细节

  4. 此外，通过将Transformer结构扩展到点生成器的基本操作来设计Upsample Transformer，它有效地结合了相邻点之间的空间和语义关系

    Transformer 可以结合相邻点空间和语义关系

1. INTRODUCTION

  1. this global feature structure 有两个内在缺陷：

    • 细粒度的细节在encoder的pooling操作中很容易丢失，并且很难从稀释的全局特征中恢复
    • 这样的全局特征是从部分点云中捕获的，因此只代表了所见部分的“不完整”信息，与生成完整形状的目的背道而驰

  2. 与预测缺失像素的 RGB 颜色的图像修复不同，3D 点生成的设计方式不同，以预测 (x, y, z) 坐标，这些坐标在 3D 空间中是非结构化但连续分布的

  3. 这表明可以描述目标对象缺失部分的期望点与其局部邻域中的周围点具有密切的空间和语义关系。

  4. 然而，现有的方法，无论是使用基于折叠的生成器 ，还是遵循具有 MLP/反卷积实现的层次结构 ，都试图通过拆分一个目标来独立处理每个点指向更多。这些设计忽略了现有点的分布，导致几何细节的恢复质量差

  5. 解决方法：

    • 首先从seed generator中的不完整特征生成完整形状
    • 然后以coarse-to-fine的方式恢复细粒度的细节

  6. patch seed 通过存储在本地种子中的学习特征来表征完整的形状结构

2. Related Work

Voxelization-based shape completion

  1. 3D 形状补全的早期尝试依赖于体素网格的中间表示来描述 3D object。这是一种将强大的 CNN 结构应用于各种 3D 应用程序的简单直接的方法，但是有信息丢失，并且计算成本在体素分辨率方面大大增加

Point cloud completion

  1. single modal: PCN is the first method: coarse to fine strategy
  2. multi-modal: view-guided point cloud completion、style-based point generator with Adversarial rendering for point cloud coompletion
  3. PoinT人： Diverse point cloud completion with germetry-aware Transformers

Point cloud generators

  1. 早期生成模型使用全连接层直接从auto-encoder中的latent representation生成点坐标
  2. FoldingNet: 通过在点变形中添加来自规范 2D 网格的变化来呈现一种新型生成器
  3. SnowflakeNet: 提出通过反卷积 (deconvolution) 拆分父点特征来生成新点
  4. 尝试Transformer

3. Method

3.1 Architecture Overview

Encoder

  1. N 是点的总数， Pi是点的坐标（x,y,z）
  2. encoder: point transformer

Seed generator

  1. Seed generator旨在预测完整形状的整体结构
  2. 给定提取的补丁特征 Fp 和中心坐标 Pp，我们使用上 Upsample Transformer 生成一组新的种子特征
  3. 通过对生成的 seed feature F 应用 MLPs 来获得相应的 seed point

coarse-to-fine generation

  1. 3.2 Point Cloud Completion with Patch Seeds

Patch Seeds

  1. Patch seeds ： seed 坐标 S 和 每个点覆盖的小区域特征 F，并且seed feature 包含这块区域的语义线索
  2. 优点：
    • 保存局部模式的区域信息
    • 通过显式监督从部分输入中恢复的完整形状结构

3.3 Upsample Transformer

Point generator via local aggregation

  1. 局部聚合
  2. 使用固定的 2D 变体独立上采样每个点，导致细节恢复不佳
  3. 解决： 基于Point Transformer，将新点的生成过程公式化为局部场中点特征的自注意力加权平均

Upsample layer

  1. following SnowflakeNet，使用前一层的输出特征作为Transformer中的key
  2. 保留输入中现有点的学习特征。然后，通过对连接的key和查询应用 MLP 来获得value
