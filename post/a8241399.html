22_CVPR_Multi-View Transformer for 3D Visual Grounding

Created2022-12-29|Updated2023-01-12|paper learningMultimodal
paper code

0. Abstract

  1. Multi-View Transformer (MVT) for the 3D visual grounding task

1. INTRODUCTION

  1. Visual Grounding (VG) aims to ground a natural language description to the target object.
  2. 之前的工作： 首先在 3D 场景中生成所有候选对象，然后选择最匹配的对象，但是这样不考虑 3D 数据的独特属性
  3. 探索学习更好的 3D visual grounding 多模态表示，这对视图变化具有鲁棒性，而不是增加视图数据或预测视图
  4. 给定特定视图下的3D场景，MVT通过以相等角度旋转原始视图将其投影到多视图空间

2. Related Work

3D Visual Grounding

  1. Solve the 3D grounding tasks following a two-stage scheme:
    • First stage: 3D object 要么由ground truth生成，要么由3D object检测器生成
    • Second stage: 3D feature 和 text feature融合后预测任务匹配的得分
  2. 现在用Transformer做特征提取和融合

Learning with Natural Language Supervision

  1. 本文利用 fine-grained object labels去加强object encoders 提高3D和text的对齐

Multi-View Learning

  1. 渲染多个视角可以获得更好的表达
  2. 与先前的不同，将 3D 场景的位置信息投影到多视图空间中，从而学习视图鲁棒表示以获得更好的视觉语言对齐

3. Method
