20_ICLR_UNPAIRED POINT CLOUD COMPLETION ON REALSCANS USING ADVERSARIAL TRAINING

Created2023-01-12|Updated2023-01-16|paper learningPoint Cloud
paper code

0. Abstract

  1. 开发了一种直接在输入点云上工作的方法，不需要成对的训练数据，因此可以直接应用于真实的扫描进行扫描补全

1. INTRODUCTION

  1. 最流行的数据驱动的扫描完成方法依赖于配对监督数据，即对于每个不完整的训练扫描，需要相应的完整数据
  2. 建立这种形状完成网络的一种方法是训练适当设计的 encoder-decoder架构（
  3. 然而，这种方法不适合难以收集大量配对监督数据的真实扫描
  4. 据我们所知，不存在 point-based unpaired method 去补全点云
  5. 传统的点云补全方法只能用一个确定的结果补全部分形状，而忽略了缺失区域的模糊性
  6. 提出了一种 unpaired point-based 的扫描补全方法，该方法不需要部分点集(例如，原始扫描)和示例完整形状模型(例如，合成模型)之间的显式对应关系
  7. 设计一个 GAN ，其中生成器，即 adaptation network，将输入转换为合适的 latent representation，让鉴别器无法区分转换后的 latent variables和从训练数据中获得的 latent variables

2. Related Work

  1. 将点云完成任务视为去噪 AE 问题，利用对抗训练来优化 AE latent space
  2. 还利用 GAN 的力量来推断部分点云扫描的缺失部分
  3. 本文的方法旨在处理未配对的数据，因此即使现实世界的和合成数据分布不同，也可以直接应用于现实世界的扫描

3. Method

  1. input: noisy and partial point set S ,target: complete point set R
