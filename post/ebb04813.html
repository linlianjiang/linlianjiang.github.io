21_Part2Word Learning Joint Embedding of Point Clouds and Text by Matching Parts to Words

Created2022-03-02|Updated2022-03-12|paper learningMultimodal
paper code

0. Abstract

Title: Part2Word: Learning Joint Embedding of Point Clouds and Text by Matching Parts to Words

  1. 学习3D形状和文本的 joint embedding
  2. 提出了一种学习点云与文本的联合嵌入方法，即在一个公共空间中，将形状中的部分与句子中的单词进行匹配
  3. 在将点云分割成部分之前，首先学习分割。然后，将部分和单词映射到一个优化的空间中，在这个空间中，部分和单词可以相互匹配。在优化后的空间中，我们通过聚合部分中所有点的特征来表示一个部分，同时用每个单词的上下文信息来表示每个单词，在这个空间中，我们训练我们的网络，以最小化三连词的排名损失

1. Introduction

  1. 将不同的3D表示映射到具有文本的joint embedding space，如体素网格和多视图
  2. 多视图下由于体素网格的低分辨率和多视图下的自遮挡导致形状信息的缺乏，使得这些方法难以提高图形和文本的联合理解能力
  3. 提出了 point-based multi-modal alignment network 来学习点云与文本的联合嵌入
  4. 点云上的part到句子的匹配
  5. 在将点云分割成部分之前，我们首先学习分割。然后，将 part 和单词映射到一个优化的空间中，在这个空间中，part 和单词可以相互匹配
  6. 在优化后的空间中，通过聚合部分中所有点的特征来表示一个部分，同时用每个单词的上下文信息来表示每个单词，在这个空间中，我们训练我们的网络，以最小化三连词的排名损失
  7. Contributions：
    • 提出了一种基于点的三维形状文本描述匹配网络框架，该框架具有语义分割的特征
    • 提出的网络在各种评价指标上实现了具有文本描述的三维形状匹配的SOTA结果

2. Related Work

2.1 Joint embedding of 3D shapes and text

直接从形状采样的点云中学习，可以更好地保留其固有的三维特性，从而获得更有鉴别性的特征

2.2 Point-based 3D deep learning

具体来说，在将点云分割成多个部分之前，我们先学习一个分割，然后将部分的点级特征进一步聚合，再与文本中的单词进行匹配

2.3 Image-text matching

现有的方法大致可以分为两类: 全局匹配和 区域匹配

Global matching methods

从图像和文本中提取全局表示，然后计算相似度得分

Regional Image-Text Matching

从已有的检测器中提取图像区域表示，然后考虑图像区域和词语层次上潜在的视觉语义对应关系

Karpathy等通过推断其跨模态对齐提出了视觉语义匹配，这些方法首先检测出目标区域，然后获得区域词对应，最后将句子中所有可能对图像区域和词的相似度进行聚合，从而推断出整体的图像-文本相似度

3. Method

  1. 受SCAN框架的启发，引入了crossattention mechanism
  2. 通过将形状中的 part 与句子中的单词进行匹配，学习三维形状与文本的joint embedding
  3. 直接利用点云三维形状的中间表示,和学习深度嵌入特性的3d点云分割获得的部分,这是一个关键的区别从先前的方法

Overview

  1. 设计了一个网络来完成 3D shape-text matching task，如上图
  2. 该网络包括三个模块: shape encoder, text encoder, and matching module
  3. 为了对三维形状 S 进行encode，使用 pre-trained segmentation network 来获得每个输入表面模型上的采样点的中间表示
  4. 然后，我们聚合这些表示来提取输入shape S 的零件嵌入 part
  5. 对于 text encoder，使用 Bi-directional Gate Recurrent Unit (GRU)学习每个单词 在 sentence 的上下文敏感嵌入
  6. 为了实现 P 和 W 之间的匹配，采用了一种alignment-based matching module（基于对齐的匹配模块），该模块利用 cross attention 对 part 与单词进行对齐，获得相似度评分
  7. 该模块包含一对对称的公式，分别表示为Shape-Text和Text-Shape

3.1 Shape Encoder

  1. 如上图所示，shape encoder 通过聚合被分割零件上对应点的特征来提取零件在每个输入形状上的嵌入
  2. 我们首先将 S 输入一个预先训练好的基于点的分割网络(在我们的例子中使用PointNet)来提取每个点的特征。
  3. 除了坐标外，还在shape encoder 中加入了每个点的颜色信息
  4. 将 PointNet 最后三层的输出 f1，f2 , f3 连接起来，以形成部分的嵌入，其中包含来自不同语义层次的信息
  5. 此外，连接了的颜色表示 color 用于利用颜色信息的输入形状
  6. 然后，将聚合后的特征和零件分割信息输入到组平均池化层中，提取每个零件的嵌入

3.2 Text Encoder

  1. 对于text encoder，使用 Bi-directional GRU来提取上下文敏感的词嵌入 W
  2. 每个文本描述 首先通过 a word embedding layer 将每个词嵌入到句子中来表示，词嵌入层中每个词的嵌入同时与网络中的其他参数进行学习
  3. 然后，在 Bi-directional GRU 中对每个单词的上下文进行编码
  4. 对于 forward GRU， 隐藏位置 i 处的状态可以从词嵌入处计算出来位置 i 和位置 i − 1 处的隐藏状态
  5. 同样，对于 reverse GRU，位置 i 处的隐藏状态由位置 i 处的词嵌入和位置 i + 1 处的隐藏状态
  6. 最后，上下文相关词嵌入由两个方向上隐藏状态的平均值

3.3 Matching

  1. shape-text 匹配模块通过 shape encoder 和 text encoder 分别提取的 part 嵌入 和上下文敏感词嵌入W，将输入的 3D形状 S 与文本 T 进行匹配
  2. part embedding 首先需要经过一个完全连接的层，以确保它与单词嵌入具有相同的尺寸
  3. 然后，引入 cross attention 来计算Shape-Text matching score, Text-Shape matching score
  4. 在Shape-Text匹配中，我们首先使用 cross attention 来建立 part 与 word 之间的关系

4. Experiments

  1. ShapeNet 和 Partnet拥有相同的 3D 模型

  2. ShapeNet包含不同的 3D 表示，包括带有颜色的点云，但没有分割标注。PartNet 包含手工标注的细粒度、实例级和分层的3D部件信息

  3. PartNet不包含三维点云的颜色信息。为了同时利用三维点云的颜色信息和部分分割标注，我们对 对齐的两个点云模型都进行了点云配准，然后我们注释分割ShapeNet 的点云上按 PartNet 上最近的带注释的相邻点添加标签

  4. 最后，我们使用11498个3D形状进行训练，1434个3D形状用于测试，包括椅子和桌子。每个3D形状平均有5个文本描述
