19_CVPR_PartNet_A Recursive Part Decomposition Network for Fine-grained and Hierarchical Shape Segmentation

Created2022-03-03|Updated2022-03-17|paper learningPoint Cloud
paper code

0. Abstract

Title: PartNet: A Recursive Part Decomposition Network for Fine-grained and Hierarchical Shape Segmentation

  1. 选择自上而下递归分解，并基于递归神经网络开发了第一个用于三维形状分层分割的深度学习模型
  2. 从表示为点云的完整形状开始，我们的模型执行递归二进制分解，其中层次结构中所有节点的分解网络共享权重
  3. 在每个节点上，训练一个节点分类器来确定其分解的类型(邻接或对称)和停止标准
  4. 在高级别节点中提取的特征递归传播到低级别节点。因此，在较高层次上的有意义的分解提供了强烈的语境线索，约束了较低层次上的分割
  5. 同时，为了提高每个节点的分割精度，我们利用提取对应部分的形状特征来增强递归上下文特征

1. Introduction

  1. Hierarchical shape segmentation （层次形状分割）通过将多类标记问题划分为级联的二值标记问题来降低形状分割的难度

  2. 另一方面，层次分割可以利用不同层次的结构约束：较高层次的分割提供强烈的线索约束较低层次的分割

  3. 如上图所示，可以精确地分割到非常细粒度的级别

  4. 基于递归神经网络(RvNN)，将三维形状分层分割成细粒度的部分。从表示为点云的完整形状开始，我们的模型执行递归二进制分解

  5. 在每个节点上，训练一个节点分类器来确定其分解的类型(邻接节点或对称节点)以及分解是否应该停止(叶节点)

  6. 因此，在较高层次上的有意义的分解约束了在较低层次上的分割。同时，为了提高每个节点的分割精度，我们利用提取的对应点云的零件形状特征增强递归上下文特征

  7. 该网络使用ShapeNet中的点采样3D模型进行训练，这些模型通常由语义标记的部件组成

  8. 对于每个形状，使用现有的基于规则的方法构建层次结构。这种有原则的训练层级有助于训练更快地收敛。在每个节点上计算损失，包括节点分类损失和二值点标记损失

  9. Contributions:

    • 基于动态RvNN架构的自顶向下分层细粒度三维形状分割的深度学习模型
    • 一种集成上下文信息和每个零件形状几何的零件特征学习方案
    • 用于3D形状的细粒度、零件实例分割的基准
    • 应用我们的细粒度结构恢复高质量的图像-形状重建

2. Related work

Learning 3D shape segmentation

基本思想是基于形状基元的几何特征来学习形状基元(如三角形、点或体素)分类器

Hierarchical segmentation of 3D shapes

  1. 3D形状通常用零件以分层结构的方式建模
  2. 层次形状分割可以通过自底向上分组方法或基于全局拓扑分析的自顶向下方式来实现
  3. 本文的方法不需要规定的规范层次结构，而是通过递归训练的节点分类，以数据驱动的方式学习分解层次结构

Recursive neural networks

3. Method

  1. 首先介绍了PartNet的总体架构，它是一个递归的部件分解网络。接下来是网络中的几个关键设计

3.1 Recursive part decomposition network

  1. 上图显示了PartNet的体系结构。以三维形状的点云作为输入，PartNet执行自顶向下的分解，并输出在零件实例级别的分割点云。在每个节点上，设计了三个模块:
    • Node decoding module（节点解码模块）用于将全局上下文信息从当前节点传递给其子节点。这些信息限制了具有更高层次上下文的节点的分割
    • Node classification module（节点分类模块设计）以构造拓扑结构的分解层次结构。这是通过学习预测节点类型来实现的，节点类型决定了如何分解节点以及何时停止分解
    • Node segmentation module（节点分割模块）用于对当前节点的点云进行实际分割。这是通过学习所有节点共享的点分类网络来实现

Node decoding module

  1. Node classification module

  2. 在给定的节点上，以其当前节点特征为输入，节点分类模块预测其节点类型为以下三种类型之一:邻接、对称或叶

  3. 该模块通过确定节点拆分的方式和是否拆分，构建层次结构的拓扑结构。该节点分类模块由两个具有双曲非线性的全连通层实现
  4. 它可以通过点云的ground-truth层次分割来训练
  5. 当一个节点被归类为对称节点时，我们将其左子节点解释为对称生成器(表示一个部分)，而其右子节点解释为对称参数
  6. 对对称产生器部分应用对称参数，得到该对称节点的完整点云。例如，与转椅腿部分(图1，左)的辐条相对应的节点为旋转对称节点。它的左子结点表示其中一个辐条的点云，而右子结点编码对称轴和对称折叠

Node segmentation module

  1. 对于对称节点，只需分割其左子节点，即对称生成器。分割后，根据预测的对称参数，将点标签转移到所有其他对称对应节点上
