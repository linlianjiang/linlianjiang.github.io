20_IEEE_Silhouette Guided Point Cloud Reconstruction beyond Occlusion

Created2022-02-17|Updated2022-03-12|paper learningPoint Cloud
paper code

0. Abstract

Title: Silhouette Guided Point Cloud Reconstruction beyond Occlusion

  1. 文章提出了一种从单个RGB图像重建物体完整3D形状的方法
  2. 给定可见区域的图像和轮廓，我们的方法完成遮挡区域的轮廓，然后生成点云
  3. 通过提供预测的完整轮廓作为指导，我们展示了非遮挡和部分遮挡对象重建的改进
  4. 还通过多个合成视图的2D重投影损失和基于曲面的平滑和细化步骤，改进了3D形状预测的最新技术

1. Introduction

  1. 推断部分遮挡物体的完整形状
  2. 采用数据驱动( data-driven )的方法，使用卷积神经网络（CNNs）对形状相关特征进行编码，并将其解码为对象点云
  3. 任务：
    • 确定物体的可见区域
    • 预测完整的轮廓（填充遮挡区域）
    • 基于轮廓和RGB图像的物体三维点云预测
  4. 以 viewer-centered manner 重建物体，推断物体的形状(shape)和姿势(pose)
  5. 文章展示了：在提供 ground truth 的情况下，形状预测在被遮挡对象和非遮挡对象中实现了几乎相同的性能
  6. 使用Mask-RCNN获得轮廓的可见部分，然后使用自动编码器预测完成的轮廓
  7. Contributions:
    • 改进了从单个RGB图像重建3D点云的最新技术。通过在多个合成视图上使用2D重投影损失和基于曲面的细化步骤，展示了性能提升
    • 完成可见的轮廓导致更好的对象形状完成。我们提出了一个轮廓完成网络，达到了最先进的水平。我们展示了重建非遮挡和部分遮挡物体的改

2. Related Work

  1. 法改进了对象点集生成的最新技术，并扩展到在完整轮廓的指导下进行遮挡以外的重建
  2. 基于预测的轮廓来解决遮挡模糊，并且能够预测完整的3D形状，而不是2.5D depth points
  3. 这项任务预测的是完整的轮廓而不是完整的纹理(texture)

3. Point Clouds from a Single RGB Image

需要一个编码器，捕捉对象的姿势和形状从单一的图像，一个解码器，能够灵活地生成无序、密集的点云

3.1 Point Cloud Reconstruction Network

  1. 网络架构如图：

    • 第一步，轮廓完成网络将RGB图像和可见轮廓作为输入，并预测前景遮挡之外的对象的完整轮廓
    • 第二步，给定RGB图像和预测的完整轮廓，重建网络在以观察者为中心的坐标中预测点云
    • 第三步，执行后细化步骤以生成平滑且均匀分布的点云

  2. 编码器基于ResNet-50，以更好地捕捉对象形状和姿势特征

3.2 Surface-based Point Clouds Refinement

  1. 一个重要的3D形状属性是光滑和连续的形状表面，特别是对于像椅子腿和轻型支架这样的薄结构(thin structures)

  2. 要强加此属性，需要细化：对密集点拟合曲面，对曲面进行平滑，对曲面进行均匀重采样，如图

    • 从左到右依次为:输入RGB图像、网络预测、FSSR曲面拟合与平滑以及从拟合曲面上重新采样的点云
    • 每个样本由相同数量的点组成，为了更好地说明，我们将每个预测的形状可视化到一个新的视图中
    • 细化步骤能够生成光滑的均匀分布的点集

  3. 通过对6个最近邻进行平面拟合来设置逐点法线的参数，并将逐点尺度设为到两个最近邻的平均距离

Smoothing

  1. 给出拟合曲面，采用隐式积分法进行平滑处理。使用曲率流作为平滑算子进行5次迭代。然后在泊松盘采样的基础上对点云进行均匀采样，得到最终的输出结果

4. Reconstruction of Occluded Objects

  1. 从RGB图像和可见区域的初始剪影开始，可以通过最近的方法(如Mask-RCNN)获得
  2. 本文提出了一个二维轮廓完成方法来生成对象的完整轮廓
  3. show：基于真实完整轮廓的预测大大提高了形状预测，并带来了对遮挡物体接近于非遮挡物体的性能;使用预测的剪影也可以提高性能，完整的剪影比可见部分的预测剪影表现更好

4.1 Silhouette Completion Network

  1. 被检测物体的RGB图像裁剪 I 和分割可见区域 SV

  2. silhouette completion network 根据 SV 预测物体完整的二维轮廓 Sf

  3. 网络采用编码器-解码器策略，以 I 和 SV 的连接作为输入，并预测Sf与Sv具有相同的分辨率

4.2 Silhouette Guided 3D Reconstruction

  1. 将预测的完整轮廓 Sf 作为一个额外的输入通道连接到输入的RGB图像 I，它可以有效地指导重建部分遮挡和非遮挡物体
  2. 提出了一种合成遮挡数据集的方法
  3. 提出了一种“cut-and-paste”策略来创建随机前景遮挡(random foreground occlusion)
  4. 从一组没有遮挡和已知的地面真实轮廓的预渲染2D图像开始，对于每个输入图像 I ，从与 I 相同的数据集拆分（训练/评估/测试）中随机选择另一个图像 I’ ，从 I’ 中删除对象段 O’ ，将 O’ 粘贴到输入图像 I
